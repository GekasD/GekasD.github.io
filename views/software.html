This is a list of free and open source software that I use and find useful.

Name Comment
Alacritty My preferred terminal emulator, it is written in rust and performs well.
Barrier Amazing internet based KVM software, very easy to set up and works like a charm with zero input delay.
Thunderbird My preferred email client.
KeePass A very simple and light-weight password manager.
GIMP My favorite and probably the best image editor.
OBS Studio The best video recording software there is.
OpenShot Simple video editor I use from time to time.
HandBrake Video transcoder I often use to shrink video sizes.
LibreOffice My preferred office Suite.
